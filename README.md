# D3-HW1

This bar chart displays the number of credit card applications by final decision using the Credit Card Applications dataset. The x-axis shows the two possible outcomes, Approved and Denied, while the y-axis represents the total number of applications in each category. The chart indicates that more applications were denied than approved, with denials totaling approximately 380 applications compared to about 310 approvals. This visualization provides a simple comparison of the overall distribution of application decisions in the dataset. The dataset used for this chart was obtained from Kaggle: https://www.kaggle.com/datasets/nazishjaveed/credit-card-application?resource=download, where it is published as the Credit Card Application dataset.
